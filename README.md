# Simple Interest Calculator

A simple interest calculator application built using Bash scripting.

## Project Description

This project provides a simple command-line tool to calculate simple interest
based on user-provided principal amount, rate of interest, and time period.

## Features

- Calculate simple interest easily
- Command-line interface
- User-friendly prompts

## Usage

Run the script using:
```bash
bash simple-interest.sh
```
